**Backend:** Node.js\
**Database:** MySQL

## Requirement

1. Flutter SDK >=2.12.0 <3.0.0
2. Android Studio 4.2 (SDK 31)

## Installation

**The app is in frontend folder**
**backend is not needed as it is deployed already**

```
cd frontend
flutter pub get
flutter run
```
